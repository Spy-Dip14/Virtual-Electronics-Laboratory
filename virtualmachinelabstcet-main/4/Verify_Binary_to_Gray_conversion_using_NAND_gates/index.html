Instructions
Step 1: Click "Check" button.
  • if connection is correct, an alert appers. Click ok. Now, go to step 2 then step 4.
  • if connection is wrong, an alert appers. Click ok. Now, go to step 3.
    • Step 2: Connect 5V battery to the completed circuit by clicking on the battery.
    Step 3: Check the connection, if incorrect, click on node number to remove any particular connection(s) or click on reset button and start from step 1.
    Step 4: Click "Add to Table" for different inputs.The corresponding combination of input & output LEDs lit up for different input combination & values get added to the truth table.
    Step 5: Click "Print" button, to print the full connections with truth table in landscape mode.
