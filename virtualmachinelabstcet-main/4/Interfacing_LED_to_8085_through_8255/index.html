Assembly Language Code

MVI A,8AH; //Load Control Word into Accumulator
OUT 83H; //Write Control Word to control port of 8255
IN 81H; //Read Input from Port B with address 81H
MOV C,A; //Move Input from Port C to register C
IN 82H; //Read Input from Port C with address 82H
ADD C; //ADD the values
OUT 80H; //Output the result to Port A with address 80H

IO Mapping of 8255 for Chip Selection
A7 A6 A5 A4 A3 A2 A1 A0
PA 1 0 0 0 0 0 0 0 80H
PB 1 0 0 0 0 0 0 1 81H
PC 1 0 0 0 0 0 1 0 82H
PA 1 0 0 0 0 0 1 1 83H

N.B: A7➡A2 determines Chip Select Logic(CS') & A1,A0 is for internal selection of address
